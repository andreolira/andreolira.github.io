CloudSync SHA256 Upgrade
1) First install the DoD Root CA #3 Certificate

2) Then install the new CloudSync mobile client for iOS or Android

3) If running iOS 10.0 or greater, go to Settings and select "General"


then select "About"


now select "Certificate Trust Settings" (note, the exact text might be different depending on operating system version)


Finally, select to trust the certificate


When prompted, select "Continue"


That's it, you're done!
